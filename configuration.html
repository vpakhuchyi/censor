Configuration

Censor provides flexible configuration options to customize its behavior. This guide covers the default configuration and how to customize it.

Default Configuration​

When you create a new Censor instance without any options, it uses the following default configuration:

c := censor.New()

Default settings:

  • Formatter: TEXT format (human-readable)
  • Mask Value: [CENSORED]
  • Exclude Patterns: None
  • Field Name Mapping: None

Custom Configuration​

You can customize Censor's behavior by providing options when creating a new instance:

c := censor.New(
    censor.WithFormatter(censor.JSONFormatter()),
    censor.WithMaskValue("[REDACTED]"),
    censor.WithExcludePatterns([]string{`\d{4}-\d{4}-\d{4}-\d{4}`}),
)

Output Formats​

Censor supports two output formats: TEXT and JSON.

TEXT Format​

The TEXT format is human-readable and is the default format:

c := censor.New(
    censor.WithFormatter(censor.TextFormatter()),
)

// Example output: {ID:123 Email:[CENSORED] Password:[CENSORED]}

JSON Format​

The JSON format is useful for structured data and API responses:

c := censor.New(
    censor.WithFormatter(censor.JSONFormatter()),
)

// Example output: {"ID":"123","Email":"[CENSORED]","Password":"[CENSORED]"}

Masking Options​

Custom Mask Value​

Change the default mask value:

c := censor.New(
    censor.WithMaskValue("[REDACTED]"),
)

Exclude Patterns​

Define patterns to exclude from masking:

c := censor.New(
    censor.WithExcludePatterns([]string{
        `\d{4}-\d{4}-\d{4}-\d{4}`, // Credit card numbers
        `\d{3}-\d{2}-\d{4}`,       // SSN
    }),
)

Struct Tag Options​

Control field visibility using struct tags:

type User struct {
    ID       string `censor:"display"` // Always visible
    Email    string                    // Masked by default
    Password string `censor:"mask"`    // Explicitly masked
}

Advanced Configuration​

Custom Type Handlers​

Register custom handlers for specific types:

type CustomType string

c := censor.New(
    censor.WithTypeHandler(reflect.TypeOf(CustomType("")), func(v interface{}) string {
        return "[CUSTOM_MASKED]"
    }),
)

Field Name Mapping​

Map struct field names to custom names:

c := censor.New(
    censor.WithFieldNameMapper(func(name string) string {
        return strings.ToUpper(name)
    }),
)

Complete Example​

Here's a complete example showing various configuration options:

package main

import (
    "fmt"
    "github.com/vpakhuchyi/censor"
)

func main() {
    // Create a Censor instance with custom configuration
    c := censor.New(
        // Use JSON formatter
        censor.WithFormatter(censor.JSONFormatter()),

        // Custom mask value
        censor.WithMaskValue("[REDACTED]"),

        // Exclude patterns
        censor.WithExcludePatterns([]string{
            `\d{4}-\d{4}-\d{4}-\d{4}`,
            `\d{3}-\d{2}-\d{4}`,
        }),

        // Custom field name mapper
        censor.WithFieldNameMapper(func(name string) string {
            return strings.ToUpper(name)
        }),
    )

    // Define a struct with sensitive data
    type User struct {
        ID       string `censor:"display"`
        Email    string
        Password string `censor:"mask"`
    }

    // Create a user
    user := User{
        ID:       "123",
        Email:    "user@example.com",
        Password: "secret123",
    }

    // Process the data
    masked := c.Process(user)
    fmt.Printf("%+v\n", masked)
    // Output: {"ID":"123","EMAIL":"[REDACTED]","PASSWORD":"[REDACTED]"}
}
